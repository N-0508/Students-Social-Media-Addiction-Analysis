# 📊 Students Social Media Addiction Analysis (EDA Project)

## 📌 Project Overview
This project performs **Exploratory Data Analysis (EDA)** on a dataset of students to understand the impact of social media usage on:

- 📱 Daily Usage Patterns  
- 🧠 Mental Health  
- 😴 Sleep Behavior  
- 🎓 Academic Performance  
- 💔 Social Conflicts  

The goal is to uncover meaningful insights and patterns that indicate **social media addiction among students**.

---

## 🎯 Objectives
- Analyze social media usage trends among students  
- Identify relationship between screen time and mental health  
- Study impact on academic performance  
- Detect signs of addiction behavior  
- Provide actionable insights  

---

## 📁 Dataset Information
- Total Records: **705 students**
- Features: **13 columns**

### 🔑 Key Features:
- Age, Gender, Academic_Level  
- Avg_Daily_Usage_Hours  
- Most_Used_Platform  
- Sleep_Hours_Per_Night  
- Mental_Health_Score  
- Conflicts_Over_Social_Media  
- Addicted_Score

## 📈 Key Insights
- 📱 Students spend **6+ hours daily** on social media  
- 😴 Higher usage leads to **reduced sleep hours**  
- 🧠 Increased usage negatively impacts **mental health**  
- 🎓 Social media affects **academic performance**  
- 💔 Higher addiction leads to more **relationship conflicts**
